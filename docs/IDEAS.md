# Идеи в рамках текущего железа (2026-06-09)

Бэклог улучшений для текущей платформы (XIAO ESP32-C6 + PCA9685 + 4× BTS7960 + JGA25-370).
По убыванию ценности. Вычёркивать по мере реализации.

## 🏎 Качество езды (чисто прошивка)
1. ~~**Ramp / плавный разгон**~~ — ✅ **сделано 2026-06-10** (50 Гц ramp-таск — единственный
   писатель PCA9685, рост ограничен/падение мгновенно; `GET/POST /ramp` + NVS; экран «Разгон»
   в iOS с демо-анимацией). Осталось: подобрать `ramp_ms` на железе слайдером.
2. ~~**Триммер прямолинейности**~~ — ✅ **сделано 2026-06-10** (`trim_apply` ослабляет быструю
   сторону; `GET/POST /trim` + NVS; экран «Прямолинейность» с демо-рельсами). Осталось: подобрать
   на железе (пустить прямо → сдвинуть слайдер против увода).
3. **Профили скорости** — «детский» 50% / «газон» 75% / «макс» 100%: лимит в прошивке +
   переключатель в аппе; экспо-кривая джойстика для точности на малых скоростях.

## ✅ Валидация уже сделанного (нужно участие у машинки)
4. **Реальный OTA-тест** — выпустить `v1.1` (аудит-фиксы уже в main) и обновиться по воздуху
   из аппа; заодно проверить rollback на битом образе.
5. **Бенч-тест watchdog** — уронить WiFi на ходу → стоп за ~300 мс (никогда не проверялся).
6. **Апп на реальном iPhone** — подпись (бесплатный Apple ID), Local Network, езда.

## 📡 Сеть / удобство
7. ~~**mDNS**~~ — ❌ **снято 2026-06-10**: был нужен только браузеру для веб-пульта (который удалён);
   iOS-апп ходит на фиксированный `192.168.4.1` (gateway softAP) — имя не требуется.
8. ~~**Веб-пульт догнать iOS**~~ — ❌ **снято 2026-06-10**: веб-пульт удалён целиком, пульт — только
   iOS-апп (`GET /` отдаёт текст-заглушку; помни про 7-дневную подпись бесплатного Apple ID).
9. ~~**Телеметрия связи**~~ — ✅ **сделано 2026-06-10** (`/status`: `rssi` с softAP + `ws_fps` +
   `wdt_trips`; палочки сигнала в аппе — по реальному RSSI с фоллбэком на пинг; «⚠ обрывов: N»
   в статус-строке при срабатываниях watchdog).

## 🔋 Когда приедет заказанное (INA260 + 3S BMS уже в пути)
10. **Мониторинг АКБ** — INA260 (V/I/P по I2C, та же шина что PCA9685) + кулоны в прошивке
    (точный SoC) → индикатор батареи в аппе. Дизайн обсуждён в сессии 2026-06-09.

## Дальний прицел (новое железо, вне этого списка)
- **ESP32-P4 + камера (FPV)** — решено: Waveshare ESP32-P4-Nano + OV5647, H.264, 480–720p@30.
  Ресёрч-отчёт в сессии 2026-06-09.
- **Лидар** — ресёрч готов: `docs/research/2026-06-10-lidar-integration.md`. Стратегия: UART-лидар
  (переживает миграцию C6→P4). Кандидаты: TF-Luna $24.90 (антистолкновение, можно уже на C6,
  UART D6/D7) + RPLIDAR C1 / LD19 (360°-радар в апп; питать через отдельный 5В DC-DC — пусковой
  ток мотора не подтверждён). На P4 видео+лидар совместимы (H.264 аппаратный), occupancy grid реалистичен.
