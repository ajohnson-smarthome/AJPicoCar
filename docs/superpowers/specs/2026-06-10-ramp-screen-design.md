# Экран «Разгон» — отдельный пункт меню — Design

**Дата:** 2026-06-10
**Статус:** дизайн утверждён (визуальный компаньон, вариант A доработанный), готов к плану

## Цель

Перенести настройку плавного старта из инлайн-секции в настройках в **отдельный пункт меню «Разгон»**
со своим экраном в общей стилистике (split-layout, как Калибровка/Прошивка): слева — машинка с
**демо-анимацией разгона**, справа — заголовок/подзаголовок/слайдер/значение.

## Пункт меню

В `SettingsView` `List` — третий `NavigationLink` после «Прошивка»:
- Иконка `gauge.with.needle` (или `speedometer`) + текст `L.rampTitle` («Разгон») — стиль как у соседних строк.
- **Инлайн-секция со слайдером удаляется** (вместе с `@State rampMs` и `.task` загрузкой из SettingsView).

## Экран `RampView` (split-layout)

### Слева — `RampCarView`: демо разгона (вариант A)
Сцена = рельсы(54) + зазор(4) + машинка(72) — **обтянута и центрируется как единое целое**
(`scaleEffect` ~1.6–1.9 как на соседних экранах). Машинка 1:1 (корпус 34×72, тёмные колёса по углам
с шевронами ▲). Демо-цикл (TimelineView, длительность = `пауза 0.4с + max(ramp_ms, 150мс) + удержание 0.8с`):
1. **Пауза** — колёса тёмные, рельсов нет.
2. **Разгон** (длится `ramp_ms`, прогресс 0→1): колёса интерполируют цвет тёмный→зелёный, шевроны
   ускоряются, рельсы растут вверх от зазора над крышей (по осям колёс) пропорционально прогрессу.
3. **Полная скорость** — колёса зелёные, рельсы полной длины, шевроны бегут.
→ снова пауза. При `ramp_ms == 0` («выкл») разгонная фаза мгновенная: пауза → сразу полная скорость.
Двигаешь слайдер → демо перезапускается с новым темпом.

### Справа — блок (единый шаблон)
- Заголовок `L.rampHeadline` («Плавный старт», 21–22pt semibold).
- Подзаголовок `L.rampSub` («Время разгона до полной мощности.», muted).
- **Слайдер** 0–1000 мс, шаг 50, `tint(accent)` (переезжает из SettingsView, та же логика:
  `POST /ramp` на отпускании, `GET /ramp` в `.task`).
- Значение: `L.rampLabel(ms)` («Плавный старт: NN мс») → показывать просто «NN мс»/«Выкл» —
  итоговые строки: `ramp.value` = "%d мс", `ramp.valueOff` = "Выкл" (заголовок уже говорит «Плавный старт»).

## Строки (`L` + `ru.lproj`)

Оставить: `ramp.title` («Разгон» — пункт меню и navigationTitle). Заменить `ramp.label`/`ramp.off` на:
```
"ramp.headline" = "Плавный старт";
"ramp.sub"      = "Время разгона до полной мощности.";
"ramp.value"    = "%d мс";
"ramp.valueOff" = "Выкл";
```

## Тестирование

- Компиляция `iphonesimulator26.2`; grep: нет остатков `rampLabel`/`rampOff`/инлайн-секции.
- Симулятор (мок): пункт «Разгон» в настройках → экран: демо-анимация крутится, слайдер показывает
  значение с мока, движение слайдера меняет темп демо, отпускание шлёт POST (лог мока). Скриншот. Обе темы.
