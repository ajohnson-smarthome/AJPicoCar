# Графические указатели мощности (сегментные LED) — Design

**Дата:** 2026-06-09
**Статус:** дизайн утверждён (визуальный компаньон), готов к плану

## Цель

Заменить текстовые указатели «Л NN% / П NN%» на главном экране на **сегментные LED-полосы** по бокам машинки —
двунаправленные, в цветах траектории (зелёный вперёд / янтарный назад). Без подписей Л/П, разнести шире от машинки.

## Компонент `PowerBar`

`PowerBar(value: Double, palette: Palette)` — `value` ∈ [−1, 1] (мощность стороны из `ControlModel.sides`).
- **5 сегментов вверх** (вперёд) · **центр-линия** · **5 сегментов вниз** (назад).
- Горят **от центра наружу** по величине: число горящих = `round(|value| · 5)`, 0..5.
- `value > 0` → зелёные (`accent`) вверх; `value < 0` → янтарные (`warn`) вниз; около нуля — все погашены.
- Сегмент: ~18×6, радиус 2, гаснущий цвет = тёмный (`#241f17`), горящий = цвет + лёгкое свечение.
- Без текста/подписей.

## `DriveView`

- В центральном `HStack` заменить `sideLabel(L.sideLeft, sides.left)` / `sideLabel(L.sideRight, sides.right)`
  на `PowerBar(value: sides.left, palette: p)` / `PowerBar(value: sides.right, palette: p)`.
- Удалить `sideLabel(_:_:)`.
- **Разнести шире:** увеличить зазор `HStack(spacing:)` (34 → ~44; подгонится по скриншоту).
- Строки `drive.sideLeft`/`drive.sideRight` («Л»/«П») и аксессоры `L.sideLeft`/`sideRight` больше не нужны — удалить.

## Тестирование

- Компиляция `iphonesimulator26.2`.
- Симулятор: idle (все погашены, видна центр-линия); форс-харнесс `DriveDiagram`-style для разных `sides`
  (вперёд → зелёные вверх обе; назад → янтарные вниз; вращение → одна зелёная вверх/другая янтарная вниз);
  скриншоты; полосы разнесены шире, без подписей. Обе темы.

## Вне объёма

- Диаграмма движения (без изменений), джойстики/схемы, прошивка.
