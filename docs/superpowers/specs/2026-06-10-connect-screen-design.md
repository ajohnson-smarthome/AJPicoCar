# Экран подключения — общая стилистика + радар — Design

**Дата:** 2026-06-10
**Статус:** дизайн утверждён (визуальный компаньон, вариант B «радар»), готов к плану

## Цель

Переработать `ConnectView` (оверлей «машинка не найдена») в общую стилистику: split-layout,
слева — тусклая машинка с **радаром** (кольца + вращающийся луч-развёртка), справа —
заголовок/подсказка/пилюля-кнопка. Заголовок становится живым: **«Поиск машинки…»**.

## Левая часть — `ConnectCarView` (радар)

- Машинка 1:1 (корпус 34×72, лобовое, тёмные колёса по углам, `palette.metal`), **притушена**
  (`opacity ~0.55`) — «её ещё нет».
- **Радар** позади машинки (Canvas + TimelineView, как у соседей):
  - 3 статичных кольца-сетки (`accent`, прозрачность ~0.16, радиусы ~46/58/70 с шагом);
  - **вращающийся луч**: сектор ~70° c angular-градиентом (`accent` 0.35 → прозрачный),
    полный оборот ~2.6 с, поверх колец, под машинкой.
- `scaleEffect` ~1.45, frame как у `RampCarView`/`TrimCarView` (120×210), композиция по центру
  левой половины.

## Правая часть

- Заголовок **«Поиск машинки…»** (≈19–22pt semibold, `text`) — строка `connect.title` меняется
  («Машинка не найдена» → «Поиск машинки…»).
- Подсказка `connect.body` — без изменений («Подключись к Wi-Fi «ESP32-Car»…»).
- Кнопка **«Открыть настройки»** — пилюля в стиле `fwButton` prominent (accent-tint + обводка),
  действие прежнее (`UIApplication.openSettingsURLString`).

  в общем стиле; вернуть мок → оверлей исчезает. Скриншот. Обе темы.
