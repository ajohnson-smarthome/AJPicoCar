# Главный экран: упорядочивание инфо-элементов + обязательная калибровка — Design

**Дата:** 2026-06-10
**Статус:** дизайн утверждён (визуальный компаньон, вариант B), готов к плану

## Цель

Расчистить главный экран: убрать справочную информацию (uptime, версия, статус калибровки) и слово
«На связи», оставить только живое (сигнал/пинг) и **проблемы** (янтарные предупреждения).
Первичная калибровка становится **обязательной** — мастер нельзя пропустить.

## Главный экран (`DriveView`)

- **Верхняя плашка:** `SignalBars` + текст **«N мс»** (`L.driveConnected` упрощается до `"%d мс"`;
  офлайн — прежнее «Поиск…»). Слово «На связи» удаляется — связь видна по палочкам.
- **Нижняя строка (`statusBar`) минимизируется:** в норме **пустая**; показывает только янтарные
  предупреждения. Единственное на сегодня: `⚠ Обрывов: N` (при `status.wdtTrips > 0`), строка
  `drive.wdtTrips` = `"Обрывов: %d"` (с заглавной). Элементы uptime / калибровка / версия удаляются
  (вместе со строками `drive.calibratedYes/No` и аксессорами; `L.uptime`-функция остаётся — нужна футеру).
- Центр (полосы+машинка), схема+⚙, джойстики — без изменений.

## Обязательная калибровка

- Шит калибровки (открывается при `calibrated == false`) становится **непропускаемым**:
  - убрать кнопку **«Позже»** из тулбара;
  - `.interactiveDismissDisabled(true)` на шите (свайпом не закрыть);
  - убрать одноразовый флаг `didPromptCalib` — шит держится открытым, **пока** `/status` не скажет
    `calibrated == true` (после успешного Save в мастере `dismiss()` как сейчас; если калибровка
    всё ещё false — шит откроется снова).
- Строка `common.later` («Позже») удаляется, если больше нигде не используется.

## Настройки: справочный футер

- В `SettingsView` под `List` — мелкая muted-строка по центру:
  **`⏱ <uptime> · ▣ <fw>`** (иконки `clock`/`cpu` как в старой статус-строке, размер ~11pt).
  Данные из `status` (уже передаётся). При отсутствии — «—».

## Компоненты

Прошивка/мок — без изменений.

## Тестирование

- Сборка `iphonesimulator26.2`; grep на удалённые ключи/кириллицу.
- Симулятор: норма — низ пустой, верх «палочки + 3 мс»; мок `wdt_trips:3` → «⚠ Обрывов: 3»;
  мок `calibrated:false` → мастер не закрывается свайпом и без «Позже»; после Save — экран езды;
  футер настроек показывает uptime/версию. Обе темы.
