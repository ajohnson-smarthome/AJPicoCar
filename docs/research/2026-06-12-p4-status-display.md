# Бортовой статус-дисплей до 4″ для ESP32-P4 — ресёрч

**Дата:** 2026-06-12 · Deep-research: 104 агента, всё high-confidence (3-0).
Задача: компактный статус-экран телеметрии на машинке (связь/RSSI, батарея, скорость, IP, калибровка),
до 4″, интерфейс любой, ESP-IDF 5.4 + LVGL, GPIO в обрез (камера+лидар+PCA9685). Плата у пользователя —
**ESP32-P4-Module-DEV-KIT**.

## Два пути

### A. Нативный MIPI-DSI — 0 GPIO ⭐ (если нужен «premium» / задел под камеру)
- DSI идёт по **выделенному D-PHY разъёму**, это **НЕ GPIO-матрица** (datasheet: DSI/CSI-пины «cannot be
  used as general-purpose I/O») → дисплей **не ест ни одного программируемого пина** (только ~1 GPIO на
  подсветку + I2C-пара на тач, обе вне DSI-линий). Идеально при занятых камере/лидаре/моторах.
- **Лучший plug-in для P4-Nano/Module:** Waveshare **3.4″ DSI Round Touch (800×800)** или **4″ (720×720)**,
  IC **JD9365**, 2-lane MIPI-DSI, 10-точечный ёмкостный тач, **~$60–70**.
- **Официально в BSP** P4-Nano: menuconfig → *Board Support Package(ESP32-P4) → Display → Select LCD type*
  есть «2.8/3.4/4inch DSI LCD» (через `waveshare/esp_lcd_dsi` + `esp_lvgl_port`) → **LVGL из коробки**,
  кастомный драйвер не нужен. ESP-IDF 5.4 нативно поддерживает DSI (`esp_lcd_new_dsi_bus`, пример
  `peripherals/lcd/mipi_dsi` с LVGL-виджетом).
- ⚠ Нужен **чип P4 rev v3.0+** и **подходящий Waveshare MIPI-DSI FPC-шлейф**. Дефолтный BSP-дисплей —
  **10.1″** (велик), малые выбираются в menuconfig.

### B. SPI TFT — дёшево и универсально ⭐ (прагматично для чистого статуса)
- Полная поддержка esp_lcd + LVGL на P4: **ST7789 (2.0″)**, **ILI9341 (2.4–3.2″)**, **ST7796 (3.5–4″)**,
  GC9A01 (1.28″ round). Официальный пример `spi_lcd_touch` (ST7789 + LVGL, двойной буфер) числит ESP32-P4
  как поддерживаемый таргет.
- **Пинов: ~5–6** (SCK/MOSI + CS + DC + RST[опц., −1] + BL). Тач для статуса **не нужен** → минимум пинов.
- На Module-DEV-KIT **28 GPIO** на гребёнке 2×20; I2C(2)+UART(2) оставляют ~24 → SPI-дисплею (5–6) с
  огромным запасом. Камера/дисплей-DSI — отдельные FFC, GPIO не едят.
- ⚠ В ESP-IDF встроены не все драйверы (ST7789 есть; GC9A01/ILI9341/ST7796 — `idf.py add-dependency` из
  Component Registry, тривиально). Цена SPI-TFT на AliExpress: **~$5–12**.

## ⚠ Не перепутать
**ESP32-P4-WIFI6-Touch-LCD-3.4C** (3.4″ round 800×800 тач) — это **готовая отдельная P4-плата с
встроенным дисплеем**, она **ЗАМЕНЯЕТ** P4-Nano, а не подключается к нему. Для «экрана к существующей
плате» бери голую панель **3.4-DSI-TOUCH-C / 4-DSI-TOUCH-C**.

## Рекомендация
- **Для бортового статуса (твоя задача) — ILI9341 2.4–3.2″ SPI** (~$6–12): дёшево, LVGL поддержан, пинов
  на Module-DEV-KIT с запасом, тач не нужен. Без тача — просто красивая телеметрия.
- **Если хочешь нативный путь / в будущем локальное превью камеры** — Waveshare **3.4″ DSI Round Touch**
  (0 GPIO, в BSP, LVGL из коробки), но дороже и нужен P4 rev v3.0+ и FPC-шлейф.

Под твою цель «статус на машинке» SPI-TFT практичнее (в разы дешевле, проще), DSI — если хочется «вау»
и переиспользовать под камеру.
